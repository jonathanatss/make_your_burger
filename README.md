# make_your-burger
Trata-se de um site de vendas de hamburguers construído com o auxílio da tecnologia VUEJS.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
